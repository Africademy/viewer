## Run Muil

To start muil editor run:

```
npm run muil
```

or

```
yarn muil
```

## Run Editor Options

```
Usage: muil [options]

Options:

-p, --port                    Editor port, default: 8000
-d, --templatesDirectory      Templates root directory, default: './templates'
```
